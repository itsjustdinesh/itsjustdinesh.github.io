Projects

PricePilot

Problem: Quick-commerce apps (Zepto, Blinkit, BigBasket) show different prices for the same product, forcing users to open several apps to compare.

Solution: Frontend price comparison demo that accepts item + quantity + location and shows sorted prices (lowest → highest). Each result links directly to the marketplace search with the query pre-filled.

Tech: HTML, CSS, JavaScript — static demo hosted on GitHub Pages. Data is editable JSON in the demo. (Scraper/API and analytics were prototyped separately in Python/C++ during the design phase.)

Impact: Demonstrates product thinking, UX-first frontend, and data display logic — great for portfolios and admissions reviewers.
