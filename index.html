Projects

PricePilot

Problem: Users waste time opening multiple quick-commerce apps (Zepto, Blinkit, BigBasket) to compare prices for everyday items — leading to confusion and missed savings.

Solution: A lightweight web demo that accepts item + quantity + location and displays sorted prices (lowest → highest) across marketplaces with direct links to each marketplace search.

Tech stack: HTML, CSS, JavaScript (vanilla). Design & UI are static and hosted on GitHub Pages. Data is demo-driven (editable JSON) — backend & scraping logic were prototyped separately using Python/C++ in the project plan.

Impact: Demonstrates product thinking, frontend engineering, and algorithmic sorting to surface actionable choices quickly — ideal for portfolios & admissions review.
